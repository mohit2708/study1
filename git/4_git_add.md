### Added files

#### Git Adding particular Files
```git
git add filename1.html filename2.html
```

#### All files added
```git
(OR) git add .
(OR) git add --all
(OR) git add -A
```

#### Add files by Wildcard
```git
git add *.java
```

#### Removing Files from the Staging Area
```git
git rm file_name.txt
(OR)
git restore file_name.txt
```
